DeathBox Assets

This page displays the SVG assets for the DeathBox AI project.

To save these assets:

  1. Right-click on the image and select "Save Image As..."
  2. Save the logo as logo.png or logo.svg
  3. Save the favicon as favicon.ico or favicon.svg

Note: For best compatibility, you may want to convert the SVG files to PNG/ICO format using an online converter.

Logo (32x32)

SVG Source: logo.svg

Favicon (16x16)

SVG Source: favicon.svg

Using these assets in your project:

For the favicon, add this to your HTML head:

<link rel="icon" href="favicon.svg" type="image/svg+xml">
<link rel="alternate icon" href="favicon.ico" type="image/png" sizes="16x16">

For the logo, you can use it in your HTML like this:

<img src="logo.svg" alt="DeathBox Logo" width="32" height="32">
